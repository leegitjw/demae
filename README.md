# 주요 기능

| 기능 | 설명 |
| --- | --- |
| 메뉴 등록, 수정, 삭제 | 메뉴 CRUD 기능 구현 |
| 스토어 등록, 수정, 삭제 | 스토어 CRUD 기능 구현 |
| 주문 생성 | 주문 생성 기능 구현 |
| 리뷰 등록, 수정, 삭제 | 리뷰 CRUD 기능 구현 |
| 인증인가 | JWT, Spring Security 활용하여 인증, 인가 구현 |
| CI/CD | 자동 배포 기능 구현 |
| AWS 이용한 주문 분리 | API Gateway, Lambda, SQS를 이용하여 주문상태 변경 로직 분리 |
